Anlisis de resultados

  1. Determinante de la matriz de correlacin: -0.00000000

    Esto es una seal de multicolinealidad severa. Un determinante cercano a cero (o negativo) significa que hay variables redundantes o muy correlacionadas entre s, al punto que la matriz es casi singular (no invertible matemticamente). No es recomendable aplicar PCA hasta hacer limpieza.

  2. VIFs infinitos (inf)

    Cuando el VIF es infinito (inf), significa que una variable se puede predecir perfectamente usando las dems variables. Esto es colinealidad perfecta, una de las peores situaciones para PCA o cualquier modelo lineal.

  3. Fallo de Bartlett (p-valor nan)

    Esto es porque hay un problema matemtico al calcular el determinante dentro del test. Si la matriz de correlacin es singular (determinante = 0 o negativo), Bartlett no se puede calcular correctamente. Por eso aparece nan.

  4. KMO = 0.5965 (Poco aceptable)

    Al tener un valor debajo de 0.6, se indica que no es recomendable aplicar PCA sin limpieza previa.

Eliminacin de variables con varianza baja (casi constantes)

Segn los resultados las variables con valor inf indica una multicolinealidad perfecta (correlacin lineal exacta con otras variables).

Anlisis de resultados obtenidos

Tras realizar una serie de pruebas estadsticas sobre las variables cuantitativas seleccionadas, se evalu si el dataset era adecuado para aplicar un Anlisis de Componentes Principales (PCA). Para ello, se realizaron las siguientes pruebas:

  1. Determinante de la matriz de correlacin
  2. Test de Esfericidad de Bartlett
  3. ndice de adecuacin muestral KMO

En primer lugar, el determinante de la matriz de correlacin result ser 0.00083177, un valor bajo que indica la presencia de colinealidad entre las variables. Este es un resultado esperado y deseado cuando se busca aplicar PCA, ya que este mtodo es ms efectivo cuando las variables tienen correlaciones significativas entre s.

Por otro lado, el Test de Esfericidad de Bartlett arroj un valor de chi-cuadrado de 10284.52 con un p-valor de 0.0. Este resultado es altamente significativo (p < 0.05), lo que permite rechazar la hiptesis nula que indica que las variables son no correlacionadas (matriz identidad). Esto significa que las variables tienen correlaciones suficientemente fuertes entre ellas.

Finalmente, el ndice KMO obtenido fue de 0.7961, lo cual se interpreta como una adecuacin muestral aceptable tirando a buena. El KMO evala si las correlaciones parciales entre pares de variables son lo suficientemente pequeas para que PCA sea apropiado. Un valor cercano a 1 es deseado, mientras que valores por debajo de 0.6 indicaran problemas de adecuacin. Con 0.7961, se puede concluir que el dataset es razonablemente adecuado para aplicar PCA.

Anlisis de resultados obtenidos mediante el anlisis de PCA

  1. Porcentaje de varianza acumulada:

    En base a los resultados obtenidos, se observa lo siguiente:

    • El primer componente (PC1) explica aproximadamente el 20.7% de la varianza total.
    • Con los primeros 5 componentes, ya se alcanza aproximadamente el 46.4% de la varianza explicada.
    • Si se consideran los 10 primeros componentes, se logra explicar el 70% de la varianza total.
    • Al llegar al componente 15, se alcanza el 89% de varianza acumulada, lo cual es coherente con la regla del 80%-90%, recomendada para retener suficiente informacin del dataset original.

    Esta tendencia muestra que el dataset es bastante complejo y disperso, lo cual requiere un nmero relativamente alto de componentes para representar adecuadamente la informacin contenida en las variables originales.

  2. Cojo de Jamb

    Al analizar la grfica de varianza explicada acumulada, es posible identificar un claro "codo" entre los componentes 15 y 17. Esto sugiere que, ms all de ese punto, agregar componentes adicionales aporta una cantidad marginal de varianza explicada, es decir, la ganancia es muy baja comparada con los primeros componentes.

  3. Conclusin

    Tomando en cuenta:

    • La regla de 80%-90% de varianza explicada.
    • El punto de inflexin observado en el Codo de Jamb.
    • La complejidad y cantidad de variables en el dataset original.

    Se decide conservar 15 componentes principales, que explican el 89% de la varianza total, permitiendo una reduccin significativa de dimensiones sin perder informacin crtica.
